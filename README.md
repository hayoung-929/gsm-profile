Express.js 기반의 RESTful API 웹 애플리케이션으로, 학생 프로필 정보를 효율적으로 관리하기 위한 백엔드 서비스를 제공합니다. 이 프로젝트는 JavaScript로 구현되었으며, 학생 데이터의 생성, 조회, 업데이트, 삭제 기능을 지원합니다.

---

## 📌 주요 기능



- RESTful API를 통한 학생 프로필 관리 (CRUD)

- 개별 학생 프로필 조회

- 학생 정보 생성 및 업데이트

- 학생 데이터 삭제

- Express.js 기반의 안정적인 백엔드 서비스

## 📖 사용법

### 실행


```bash
npm start
```

개발 서버가 시작되어, 정의된 RESTful API 엔드포인트들을 사용할 수 있게 됩니다. 일반적으로 `http://localhost:3000`에서 서비스됩니다.



### 사용 예시


```bash
모든 학생 프로필 조회: `GET /students` 엔드포인트로 요청하여 전체 학생 목록을 가져옵니다.
```

```bash
새로운 학생 프로필 생성: `POST /students` 엔드포인트에 JSON 본문을 전송하여 새로운 학생을 추가합니다.
```

```bash
특정 학생 프로필 조회: `GET /students/{studentId}` 엔드포인트로 요청하여 특정 ID를 가진 학생의 정보를 가져옵니다.
```
